V2017.06.20
    1、这是一个Delta机械臂，由大族H7舵机构成。它要完成的功能是进行
变速地灰度雕刻功能。

V2017.07.05
    1、频率应该设置为f<=125Hz，如果f>125Hz,在绘制的过程中会出现暂留点的情况。
    2、主要参数是：
        f：频率，表示目标位置的更新速度
        unit：表示解析路径的精度，如果unit=0.1, 表示以0.1mm为单位分割路径。当unit越大时，
        repeatNum：重合度，表示最终解析出的目标刻度完全重合的个数，它在一定程度上影响了绘制过程的
    精度和速度。例如：repeatNum=3,表示每个目标刻度组重复3次。unit越小，repeatNum越大。例如：
    当unit = 0.01时，repeatNum=15；当unit = 0.06时，repeatNum=2。
        暂留点：由于f过大(一般f>=125Hz)时，在运动的过程中会造成，机械臂运动一会儿停一会儿又运动
    一会儿的奇怪现象。频率越大，暂留点出现的越频繁。
    3、几个常用的参数：
        3.1、绘制半径小于1mm的图形时，f=125Hz，unit<=0.01，这样才能绘制精细部分
        3.2、当雕刻比较大的图形时，为了保证速度，unit可是设置大一些，例如：unit=0.06,但是要
    注意当unit=0.06时, repeatNum=2, 如果unit超过0.06可能会造成损失路径点，也就是说，unit越大
    路径点越小，从而机械臂的运动速度会越快，同时图形也会越失真。
        3.2、推荐的参数设置为：f=125Hz, unit = 0.05;
